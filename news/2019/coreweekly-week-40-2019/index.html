PrestaShop Core Weekly - Week 40 of 2019

An inside look at the PrestaShop codebase

Written by
PrestaShop team
Published

Nov 5, 2019

Share this

This edition of the Core Weekly report highlights changes in PrestaShop’s core codebase from Monday 30th of September to Sunday 6th of October 2019.

A quick update about PrestaShop’s GitHub issues and pull requests:

  • 80 new issues have been created in the project repositories;
  • 78 issues have been closed, including 18 fixed issues on the core;
  • 61 pull requests have been opened in the project repositories;
  • 71 pull requests have been closed, including 63 merged pull requests.

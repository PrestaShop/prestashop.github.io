PrestaShop Core Weekly - Week 23 of 2020

An inside look at the PrestaShop codebase

Written by
PrestaShop team
Published

Jun 9, 2020

Share this

This edition of the Core Weekly report highlights changes in PrestaShop’s core codebase from Monday 1st to Sunday 7th of June 2020.

A quick update about PrestaShop’s GitHub issues and pull requests:

  • 104 new issues have been created in the project repositories;
  • 75 issues have been closed, including 17 fixed issues on the core;
  • 122 pull requests have been opened in the project repositories;
  • 94 pull requests have been closed, including 77 merged pull requests.

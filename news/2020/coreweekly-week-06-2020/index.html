PrestaShop Core Weekly - Week 6 of 2020

An inside look at the PrestaShop codebase

Written by
PrestaShop team
Published

Feb 11, 2020

Share this

This edition of the Core Weekly report highlights changes in PrestaShop’s core codebase from Monday 3rd to Sunday 9th of February 2020.

A quick update about PrestaShop’s GitHub issues and pull requests:

  • 64 new issues have been created in the project repositories;
  • 74 issues have been closed, including 16 fixed issues on the core;
  • 54 pull requests have been opened in the project repositories;
  • 52 pull requests have been closed, including 39 merged pull requests.

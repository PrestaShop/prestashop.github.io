PrestaShop Core Weekly - Week 37 of 2022

An inside look at the PrestaShop codebase

Written by
PrestaShop team
Published

Sep 19, 2022

Share this

This edition of the Core Weekly report highlights changes in PrestaShop’s core codebase from Monday 12th to Sunday 18th of September 2022.

A quick update about PrestaShop’s GitHub issues and pull requests:

  • 34 new issues have been created in the project repositories;
  • 37 issues have been closed, including 10 fixed issues on the core;
  • 63 pull requests have been opened in the project repositories;
  • 59 pull requests have been closed, including 49 merged pull requests.

PrestaShop Live Update - February 2023

PrestaShop integration with Vue Storefront, faceted search module, and a new product page presentation!

Written by
Krystian Podemski
Tech Evangelist @ PrestaShop
Published

Feb 14, 2023

Share this

The project’s live update was held for the second time this year and streamed live on February 8th, 2023.

Demos that you don’t want to miss!

Safe to say, this was one of the best, if not the best, live updates we ever held. During February’s event, we had a chance to see one of the most popular PrestaShop community projects. Integration between PrestaShop and Vue Storefront created by BinShops.

That was not all! The following demo about the faceted search module presented by Daniel Hlavacek (@Hlavtox) was something the community wanted to see for a long time. Starting with version 3.11, you can filter products not only on the category pages but also on new products, price drops, and bestsellers. There is also support for filtering search results which is now in the experimental phase.

Last but not least, the new product page overview, coming to PrestaShop 8.1, was like the cherry on top. It was created from scratch with hundreds of improvements, new features, and bug fixes. This new page addresses merchants’ feedback: revamped combinations management, a new way of working with multi-store, and much more.

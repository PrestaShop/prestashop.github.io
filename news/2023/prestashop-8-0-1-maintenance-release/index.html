PrestaShop 8.0.1 is available

Maintenance version of the 8.0.x branch

Written by
PrestaShop team
Published

Jan 6, 2023

Share this

The first patch version for PrestaShop 8 is now available to download!

After a few weeks of intensive work from all project members, the first maintenance release for version 8 is now available.

In this patch, you can find more than 30 bug fixes and one performance improvement that prevents clearing the system’s cache for bulk actions performed from the modules list. You will also get changes from the recent PrestaShop patch release 1.7.8.8.

It is recommended to upgrade your shop quickly in order to benefit from these fixes. Of course, don’t forget to backup before.

Reminder: by the time this article is published, the 1-Click Upgrade module’s latest version is v4.15.0, don’t forget to upgrade it.

Download PrestaShop 8.0.1 now!

Since version 8.0.1 is a “patch” update, upgrading from version 8.0.0 should happen without any issues.

Features will work better, and modules and themes, which worked fine on 8.0.0, will work just as well with 8.0.1. As for every upgrade, it’s highly recommended to do a full manual backup beforehand.

If you encounter any problems during the upgrade that, in your opinion, are not caused by third-party modules or modifications used in your shop, submit a bug report.

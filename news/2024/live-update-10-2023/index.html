PrestaShop Live Update - December 2023

A summary of what happened during the December 2023 Live Update

Written by
Krystian Podemski
Tech Evangelist @ PrestaShop
Published

Jan 3, 2024

Share this

The project’s live update was held for the tenth time this year and streamed live on December 20th, 2023.

PrestaShop Live Update - December 2023

The first Live Update after a month-long break, due to the PrestaShop Developer Conference, was the occasion for an important announcement, which probably comes as no surprise now that you’re reading this blog. The project now has the same branding PrestaShop SA has been using for several months. We also had a chance to see a keynote from @matks about the project’s accomplishments in 2023 and plans for the next major release, version 9.0.0. And also a look at the new PrestaShop API, thanks to the presentation from @jolelievre.

The session’s agenda was as follows:

  • 00:00 Beginning of the session: the introduction
  • 01:31 Project news: New PrestaShop Project branding
  • 02:34 Project news: Hooks in Hummingbird theme. New project created by @ElodieOS
  • 05:45 Project news: PrestaShop Developer Conference
  • 07:25 Demo: PrestaShop News. Where we are, and where we are going, by @matks
  • 20:15 Demo: API OAuth/CQRS in PrestaShop 9, by @jolelievre
  • 59:21 Q&A session

You can use the timestamps to jump directly to the parts you want to watch.

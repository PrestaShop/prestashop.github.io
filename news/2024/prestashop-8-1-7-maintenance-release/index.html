PrestaShop 8.1.7 Is Available

New patch for branch 8.1.x is available

Written by
PrestaShop team
Published

Jun 25, 2024

Share this

A new patch for PrestaShop 8.1 is now available! This release includes several significant improvements and numerous bug fixes. In this article, we’ll explore these updates in detail.

PrestaShop 8.1.7 is now available. This release addresses several bugs and brings various improvements.

It is recommended to upgrade your shop quickly in order to benefit from these fixes. Of course, don’t forget to backup before. Upgrade is possible using the latest version of the 1-Click Upgrade module.

Let’s dive into the details of this release.

Notable changes in version 8.1.7

Check out some of the most notable improvements and bug fixes from this version.

Bug fixes

There are a few bug fixes in this version that are worth mentioning.

  • @Progi1984 backported a fix from @unlocomqx to fix the issue with the quantity check when there are customized products in a cart. (#36342).
  • @Hlavtox fixed the issue where multiple carriers are used and a package does not have an assigned carrier. (#36074).
  • @rudolfkastl fixed a problem with the credit slip creation. (#35861).

You can see the rest of pull requests with bug fixes on Github.

Improvements

In version 8.1.7, you will find several improvements. Some of the most notable ones are:

  • Thanks to @webeshop, when you duplicate a product, the product date is now correctly set to the current date. (#34823)
  • The export button is now available on the product page. Thank you, @ChronoBrake. (#35488).
  • @mattgoud has improved the product page responsiveness. (#35775).
  • The wording related to managing pack products has been improved by @Hlavtox. (#35434).
  • The field size for meta_title and meta_description has been corrected by @M0rgan01. (#35564).
  • @jolelievre fixed miscellaneous bugs on the new product page related to combinations and features management. (#36244).

Improvements above should make your daily work with managing products easier and more efficient.

Apart from the ones mentioned above, there are other notable improvements in this version. One worth mentioning is the cart rules performance improvement. You can check for more details in the pull request by @ShaiMagal. Also, to make life easier for developers, the EntitySearchInput component is now available in the window.prestashop.component thanks to @thomasDelaporte.

Make sure to check the full list here.

PrestaShop 8.1.7 is now available for you

Fixes and improvements mentioned above are just a part of all changes in this release. You can find the full list of changes on the release page, you can also check the list of all merged pull requests in this release.

Download PrestaShop 8.1.7 now!

Since version 8.1.7 is a “patch” update, upgrading from previous 8.1.x versions should happen without any issues. As for every upgrade, it’s highly recommended to do a full manual backup beforehand.

If you encounter any problems during the upgrade that, in your opinion, are not caused by third-party modules or modifications used in your shop, submit a bug report.

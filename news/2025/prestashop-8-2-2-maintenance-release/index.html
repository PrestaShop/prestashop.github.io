    PrestaShop 8.2.2 is available

    The second patch for branch 8.2.x is available

    Written by
    PrestaShop team
    Published

    Jul 28, 2025

    Share this

    PrestaShop 8.2.2 is here! This latest patch version brings a range of updates, addressing reported issues and introducing enhancements to improve the platform for merchants and developers alike.

    We are happy to announce the release of PrestaShop 8.2.2, a patch version that not only resolves numerous bugs reported by our dedicated community and Quality Assurance team but also introduces some improvements.

    As part of the Call for Contributions challenge, PrestaShop 8.2.2 includes several new hooks contributed by the community. These hooks open up more possibilities for customization and integration. You can explore the list of newly added hooks here, we also briefly describe them below.

    What’s new in PrestaShop 8.2.2?

    This release includes a variety of improvements and bug fixes across different areas of PrestaShop, including the back office, front office, core, installer, and web services. Here are some highlights:

    Back Office

    New Features

    • A new hook actionProductGetAttributesGroupsAfter has been added, enabling developers to manipulate attribute groups after product data is fetched. Learn more.
    • Two new hooks, actionOrderHasBeenShipped and actionOrderHasBeenDelivered, have been introduced to enhance order tracking capabilities. Details here.

    Improvements

    • The HelperCard now uses v8 documentation for better alignment with the current version. More info.
    • Improved regex in StringModifier for better string handling. See changes.
    • Autofill of passwords has been disabled when editing customer details in the back office. Read more.
    • Mail template directories are now checked before reading, preventing errors. Details.
    • Improved tax rule selection query for better performance. See changes.

    Bug Fixes

    • Support added for brands named as numbers. Fix details.
    • Corrected misleading maximum attachment upload size in advanced settings. Learn more.
    • Resolved error related to displaying the product price in back office in some scenarios. See fix.
    • Fixed the configuration for maximum file size in the product form. Details.
    • Addressed issues with translations saving process in the back office. More info.
    • Resolved errors with CMS page creation in multistore setups. Fix details.
    • Corrected password policy to align with bcrypt limitations. Learn more.
    • Fixed refund functionality for orders linked to deleted customers. Details.

    Front Office

    New Features

    • Added displayCartExtraProductInfo hook to display additional product information in the cart. Learn more.
    • Introduced actionCartDuplicate hook for duplicating cart actions. Details here.
    • New hook actionPaymentModuleProductVarTplAfter added for payment module customization. More info.

    Bug Fixes

    • Modules returning empty payment options are now filtered out. Fix details.
    • Address fields now remain visible after changing the country on the checkout page. Learn more.
    • Corrected multilingual translation issues. Details.

    Core

    Improvements

    • Two new hooks added in getImageLink for enhanced image handling. Learn more.
    • Backported improved profiler from version 9 for better debugging. Details here.

    Bug Fixes

    • Fixed rounding calculations in the price calculator. See fix.
    • Corrected SK tax in localization files. Details.

    Installer

    Improvements

    • Added demo product environment parameter for CLI installation via Docker. Learn more.

    Web Services

    Bug Fixes

    • Resolved SQL error in web services. Fix details.
    • Fixed issues with product carrier deletion in web services. Details.

    PrestaShop 8.2.2 brings these updates and more to enhance your experience. For a complete list of changes, visit the release page.

    Update to PrestaShop 8.2.2

    We strongly encourage you to update your shop to PrestaShop 8.2.2 promptly to benefit from the extensive bug fixes and enhancements included in this release. Prior to updating, make sure to create a full backup of your shop to safeguard your data.

    Updating to the latest version of PrestaShop 8.2.x is straightforward with the Update Assistant. This tool streamlines the update process, ensuring your shop remains current with minimal effort and maximum reliability.

    As highlighted in our recent article (PrestaShop 8.2.x enters the extended support phase), version 8.2.x will now only receive security patches and critical updates. To take advantage of the latest features and enhancements, we recommend starting your migration plan to PrestaShop 9.

    If you encounter any problems during the update that, in your opinion, are not caused by third-party modules or modifications used in your shop, submit a bug report.
